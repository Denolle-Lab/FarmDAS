# Fiber-optic sensing for soil hydrodynamics under farming practices

This package uses ambient noise interferometry to monitor soil's 
hydrodynamic response under precipitation, evapotranspiration, and
drainage, assessing the soil degradation under tillage and compaction.

Data processing is based on the pacakge [Noisepy4DAS][noisepy4das].
To reproduce figures in [our latest study][paper], you can refer to
[jupyter notebooks][jupyters].




[noisepy4das]: https://github.com/niyiyu/NoisePy4DAS-SeaDAS
[paper]: https://arxiv.org/abs/2509.09821
[jupyters]: https://github.com/Denolle-Lab/FarmDAS/tree/main/dvv_and_model
