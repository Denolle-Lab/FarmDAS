# Fiber-Optic Sensing for Soil Hydrodynamics under Farming Practices

[![License: CC BY-NC 4.0](https://licensebuttons.net/l/by-nc/4.0/88x31.png)](https://creativecommons.org/licenses/by-nc/4.0/)


This package uses **ambient noise interferometry** to monitor soil hydrodynamic responses under **precipitation, evapotranspiration, and drainage**, allowing the assessment of soil degradation caused by **tillage and compaction**.

## Features
- Analyze soil water fluxes using distributed acoustic sensing (DAS).  
- Process and interpret soil responses to agricultural practices.  
- Reproduce figures from our latest study using provided Jupyter notebooks.

## Dependencies
Data processing is based on the package [NoisePy4DAS][noisepy4das].

## Reproducing Figures
To reproduce figures from [our latest study][paper], see the provided [Jupyter notebooks][jupyters].
